
# テキストアノマライザー

**テキストアノマライザー** は、文章中の「人名・社名・メールアドレスなど」を  
指定した置換マスター(JSON)をもとに匿名化（マスキング）できる、  
シンプルなクライアントサイドWebツールです。

[👉 ツールを開く (GitHub Pages)](https://Rutile3.github.io/text-anonymizer/)

---

## 🧩 概要

このツールは、ChatGPTなどのAIサービスや公開フォーラムにテキストを送る前に、  
**固有名詞を安全に伏せる** ための前処理支援ツールです。  

自動検出は行わず、  
**ユーザーが定義した置換マスター(JSON/表)** に基づいて確実に変換します。

すべての処理はブラウザ上で完結し、  
データは一切サーバに送信されません。

---

## ✨ 主な特徴

- 🧱 **4分類のマスター編集**
  - 人名・社名・メールアドレス・その他  
  - ブラウザ上で直接編集できる表形式UI
- 💾 **localStorage保存**
  - ルールはブラウザに保存され、次回もそのまま利用可能
- 📤 **JSONインポート／エクスポート**
  - 他の環境への共有やバックアップも簡単
- 🕳️ **マスキングモード**
  - 元の長さを維持して「○」で伏せるオプションあり
- 🧠 **英数字の単語境界置換**
  - 英文やコードに対して誤置換を抑制
- 🪶 **軽量＆静的**
  - GitHub Pagesでホスティング可能なHTML/CSS/JSのみ構成

## 🧰 使い方

1. **GitHub Pagesを開く**  

   → [https://Rutile3.github.io/text-anonymizer/](https://Rutile3.github.io/text-anonymizer/)

2. **右ペインで置換マスターを編集**
   - 「大分類」「置換前」「置換後」を追加して保存します。
   - 例：  
  
     | 大分類 | 置換前 | 置換後 |
     |----------|---------|---------|
     | 人名 | 山田太郎 | A会社のYさん |
     | 社名 | 株式会社テスト | △△社 |
     | メール | taro.yamada@example.com | ***@example.com |
     | その他 | 大阪城 | 某所 |

3. **入力エリアにテキストを貼り付け**
   - 伏せたいテキストを入力欄に入れ、「置換を実行」をクリック。

4. **結果を確認／コピー**
   - 出力欄に置換後のテキストが表示されます。
   - 「コピー」または「ダウンロード」で取得可能。

## ⚠️ 注意事項

- 本ツールは「ユーザー定義ルール」に基づいて置換を行うものです。  
  **自動で人名や社名を判定するAI機能は搭載していません。**
- 変換後のテキストは、必ず人間の目で最終確認を行ってください。
- すべての処理はローカルで実行され、入力データは送信されません。
